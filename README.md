Para executar (aplicação está disponível em deploy), execute sequencialmente:
```bash
git clone https://github.com/niveaabreu/Tiki-Taka-Backend

python3 -m venv env

cd env/Scripts 

activate 

cd ../..

pip install -r requirements.txt


python manage.py runserver
```
